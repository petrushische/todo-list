# Todo-List

## Функционал

Добавление задач, отмечать как выполненные, удалчение задач, фильтрация по задача (выполненные, активные). Сохранение в Local Storage

## Технологии

В данной работе используется :

- Верстка HTML
- Стили Css
- JS
